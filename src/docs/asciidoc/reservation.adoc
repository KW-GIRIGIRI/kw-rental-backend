=== 현재 내 대여 예약 조회 API

각 대여 예약 상세의 상태에 대한 해석

- `RESERVED` : 대여 신청
- `RENTED` : 대여 중
- `RETURNED` : 반납됨
- `ABNORMAL_RETURNED` : 비정상 반납(파손, 분실, 연체 반납)
- `OVERDUE_RENTED` : 연체 중
- `CANCELED` : 대여 취소
